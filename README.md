Python Programming Lab

Aim of the Lab
The aim of this lab is to practice fundamental Python programming concepts including:
 Conditional statements (`if-else`, `if-elif-else`)
 Loops (`for`, `while`)
 Functions
 Menu-driven programs
 Basic input/output operations

 List of Programs

1. Check whether a number is positive, negative, or zero
2. Check whether a number is even or odd
3. Find the largest of two numbers
4. Find the largest of three numbers using nested if-else
5. Display grade based on marks
6. Menu-driven calculator (addition, subtraction, multiplication, division)
7. Print numbers from 1 to 10 using a for loop
8. Print all even numbers from 1 to 20
9. Find the sum of the first 10 natural numbers
10. Display the multiplication table of a number
11. Print numbers from 1 to 5 using a while loop
12. Calculate sum from 1 to n using a while loop
13. Take input continuously until user enters 0
14. Print numbers from 1 to 10 but stop at 6 using break
15. Print numbers from 1 to 10 but skip 5 using continue
16. Search for a number in a list and stop when found
17. Print “Hello, World!” using a function
18. Return the square of a number using a function
19. Add two numbers using a function
20. Check whether a number is even or odd using a function
21. Calculate factorial of a number using a function
22. Print multiplication table using a function
23. Print numbers from 1 to n and display even/odd using a function and loop
24. Menu-driven calculator using functions and loop until exit

## Brief Explanation of Each Task

1. **Check positive/negative/zero** – Takes input and prints if the number is positive, negative, or zero.  
2. **Check even/odd** – Uses modulus operator `%` to determine parity.  
3. **Largest of two numbers** – Compares two numbers using `if-else`.  
4. **Largest of three numbers** – Uses nested `if-else` to find the largest among three numbers.  
5. **Display grade** – Assigns grades (A, B, C, Fail) based on marks.  
6. **Menu-driven calculator** – Performs addition, subtraction, multiplication, and division until user exits.  
7. **Print 1 to 10** – Uses a `for` loop to print numbers from 1 to 10.  
8. **Print even numbers 1–20** – Uses a loop and `% 2` check to print even numbers.  
9. **Sum of first 10 natural numbers** – Calculates the sum using a `for` loop.  
10. **Multiplication table** – Prints multiplication table of a given number.  
11. **Print 1 to 5 using while loop** – Uses a `while` loop to iterate from 1 to 5.  
12. **Sum from 1 to n** – Calculates sum from 1 to user-input `n` using a `while` loop.  
13. **Continuous input until 0** – Repeatedly takes input until the user enters 0.  
14. **Print 1–10 stop at 6** – Demonstrates `break` statement to exit a loop early.  
15. **Print 1–10 skip 5** – Demonstrates `continue` statement to skip an iteration.  
16. **Search in list** – Loops through a list and stops when the target number is found.  
17. **Print “Hello, World!”** – Defines a simple function to print text.  
18. **Square of a number** – Function that returns the square of a number.  
19. **Add two numbers** – Function that returns sum of two numbers.  
20. **Check even/odd using function** – Uses a function to determine parity.  
21. **Factorial using function** – Calculates factorial of a number using a loop inside a function.  
22. **Multiplication table using function** – Prints the multiplication table for a given number.  
23. **Print numbers 1–n with even/odd** – Uses a loop inside a function to display numbers with their parity.  
24. **Menu-driven calculator using functions** – Fully functional calculator with addition, subtraction, multiplication, and division in a loop until the user exits.
